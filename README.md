# 🌿 Greenden — Responsive Plant Store Web Application

Greenden is a modern, responsive front-end web application designed to simulate an online plant store. The project focuses on clean UI design, responsive layouts, and structured component-based development using Tailwind CSS.

---

## 📌 Overview

This project demonstrates the implementation of a multi-page website using semantic HTML and utility-first CSS. It provides a visually appealing interface for browsing plant products and understanding basic e-commerce layout principles.

---

## ✨ Key Features

* **Responsive Navigation Bar** with mobile-friendly layout
* **Landing Page (Home)** with hero section and product highlights
* **Products Page** with search interface and grid-based layout
* **Customer Review Section** for user feedback representation
* **Newsletter Subscription UI**
* **Consistent Design System** using Tailwind CSS utilities
* **Interactive UI Elements** with hover transitions and animations
